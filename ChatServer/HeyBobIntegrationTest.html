🍚 HeyBob 통합 테스트

Main Server ↔ Chat Server 연동 및 실시간 메시징 테스트

Main Server: ❓
Chat Server: ❓
채팅방: 방 ID를 입력하세요

📋 테스트 가이드

  1. 1단계: 두 사용자 모두 "연결" → "구독" 버튼 클릭
  2. 2단계: 메시지 입력으로 일반 채팅 테스트
  3. 3단계: "💰 정산 요청", "✅ 정산 완료", "🏦 적금 요청", "✅ 적금 완료" 버튼으로 Main Server 연동 테스트
  4. 4단계: "🍚 학식 정보" 버튼으로 Chat Server 내부 기능 테스트
  5. 5단계: "🤖 AI 봇" 버튼으로 AI 메뉴 추천 기능 테스트

💡 중요: Main Server(8080), Chat Server(8081) 모두 실행 필요
⚠️ 금융 메시지(정산/적금 요청/완료)는 Main Server를 통해서만 전송됩니다
✅ 일반 채팅, 학식 정보, AI 봇은 Chat Server에서 직접 처리됩니다
🤖 AI 봇은 OpenAI API 키가 필요합니다 (application.yml 설정)
